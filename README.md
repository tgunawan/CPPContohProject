# CPPContohProject
Berisi semua hasil dan contoh latihan CPP dan buildnya

## Ringkasan isi repositori
- Berisi banyak contoh program C++ di root, termasuk latihan fungsi, loop, file handling, OOP, dan modul lain.
- Folder `Project C++/Semester 1` menyimpan beberapa proyek semester 1 seperti calculator, game tebak angka, dan layanan kamar.
- File build/hasil kompilasi seperti `*.exe` saat ini ada di repo, dan `.gitignore` telah ditambahkan untuk mengabaikan file binari dan folder build ke depan.

## Catatan
- Update `.gitignore` jika dibutuhkan pola tambahan nanti.
- Gunakan branch `main` yang terhubung ke remote GitHub `origin`.
